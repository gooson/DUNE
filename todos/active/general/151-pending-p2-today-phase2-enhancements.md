# Today Tab Phase 2: Intermediate Enhancements

## Scope

docs/brainstorms/2026-03-29-today-tab-evolution.md Phase 2 항목:

### 2-A. Adaptive Hero Message (적응형 히어로)
- 시간대+상태별 히어로 메시지 변신
- CoachingEngine 확장: 06-10, 12-14, 18-22, 22+ 시간대별 메시지

### 2-B. Quick Actions Row (빠른 동작)
- Hero 아래 가로 스크롤 Quick Actions
- 운동 시작, 수면 기록, 체중 기록, 자세 촬영
- QuickStartPopularityService 활용 순서 자동 정렬

### 2-C. Daily Progress Ring (3대 미니 링)
- 활동(걸음수/목표), 수면(어젯밤/목표), 습관(완료율) 3대 링
- Apple Watch Activity Rings 스타일

### 2-D. Exercise Intelligence Card
- 투명한 추천 근거 카드
- 추천 이유를 근육 피로, 미사용 일수, 컨디션, 수면으로 설명

### 2-E. Yesterday Recap Card
- 오전(6-12시)에만 표시
- 어제 운동, 수면, 컨디션 변화 한 줄 요약
