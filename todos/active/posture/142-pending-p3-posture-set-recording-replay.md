# Phase 4C: 세트 녹화 & 리플레이

## 설명

운동 세트 전체를 비디오 + 관절 데이터로 녹화하고, 리플레이 시 구간별 분석 제공.

## 주요 작업

1. **녹화 시스템**
   - 비디오 녹화 (AVAssetWriter)
   - 관절 데이터 사이드카 (타임스탬프별 JSON)
   - 동기화된 타임라인

2. **리플레이 뷰**
   - 비디오 재생 + 스켈레톤/각도 오버레이 동시 렌더링
   - 슬로우모션, 프레임 단위 이동
   - 특정 렙 구간 점프

3. **구간별 점수**
   - 렙별 점수 산출
   - 최악 구간 자동 하이라이트
   - 세트 전체 요약 (평균 점수, 피로 추세)

4. **세트 간 비교**
   - 같은 운동 이전 세트 vs 현재 세트
   - side-by-side 또는 오버레이 비교

## 착수 조건

- Phase 4B 완료 (폼 판정 안정)

## 검증 기준

- [ ] 녹화-리플레이 타임라인 동기화 오차 < 1 프레임
- [ ] 리플레이 시 스켈레톤 오버레이 정확 매핑
- [ ] 렙별 점수가 실시간 판정과 일치
- [ ] 저장 용량: 1분 세트 < 50MB
