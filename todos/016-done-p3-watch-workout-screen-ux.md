# Watch 운동 중 화면 UX 개선

## 현재 상태

MetricsView, SetInputSheet, ControlsView 등에 DS 토큰이 적용되었으나,
정보 계층화와 인터랙션 개선은 미완료.

## 목표

- MetricsView: 현재 세트/총 세트 진행 표시기 (프로그레스 링 또는 바)
- SetInputSheet: 이전 세트 기록 표시 (히스토리 참조)
- RestTimerView: 세트 간 자동 전환 UX 개선
- SessionSummaryView: 운동별 볼륨 breakdown 시각화
