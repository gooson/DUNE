# Watch 운동 선택 타일 레이아웃 개선

## 현재 상태

QuickStartPickerView가 ExerciseTileView 컴포넌트를 사용하지만, 여전히 단순 리스트 형태.
브레인스톰에서 계획한 Apple Health 스타일 시각적 타일 UI는 미구현.

## 목표

- 카테고리별 그룹핑 (근력, 유산소, 유연성)
- 최근 운동 카드 섹션 (큰 타일, 마지막 기록 표시)
- 전체 운동 리스트는 카테고리 섹션으로 분리
- 검색 기능 유지
