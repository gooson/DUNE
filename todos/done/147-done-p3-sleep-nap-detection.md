# 낮잠 감지 및 분리

## 설명

30분 이상의 낮잠을 자동 식별하고 야간 수면과 분리하여 추적한다.

## 상세

- HealthKit 수면 데이터에서 주간(06:00-20:00) 수면 세션 감지
- 30분 이상 지속된 세션을 낮잠으로 분류
- 낮잠은 Sleep Score 계산에서 제외 (야간 수면만 반영)
- 별도 "낮잠 기록" 섹션에서 빈도, 평균 길이 추적
- 낮잠과 야간 수면 품질의 상관관계 인사이트

## 선행 조건

- 없음

## 엣지 케이스

- 교대 근무자의 주간 수면은 낮잠이 아닌 주 수면으로 처리 필요
- 사용자 설정으로 "주 수면 시간대" 커스터마이즈 고려
