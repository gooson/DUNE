# BlendShape 볼륨 애니메이션

## 설명

볼륨 데이터(주간 세트 수)에 따라 근육 메시가 팽창/수축하는 morph target 애니메이션 구현.
RealityKit BlendShape 지원 확인 필요 (iOS 17+).

## 선행 조건

- USDZ 메시 교체 완료 (MVP)
- Blender에서 각 근육군별 BlendShape key 생성
