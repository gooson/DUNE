# Fun Comparison Enhancement

## Description

현재 월간 합산 기준 fun comparison을 확장: 전체 기간 누적 비교, 계절별 비교, 다양한 비교 템플릿 추가.

## Scope

- 누적(all-time) 비교 옵션 추가
- 비교 템플릿 다양화 (음식, 동물, 거리, 건물 등)
- 로케일별 비교 대상 (한국: 서울-부산, 일본: 도쿄-오사카)
- 월별 변화 추이 ("지난달 대비 30% 증가")
