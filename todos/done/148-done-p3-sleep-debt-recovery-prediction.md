# 수면 부채 회복 예측

## 설명

현재 수면 부채 기반으로 정상 수준 회복까지 예상 소요일수를 계산한다.

## 상세

- 기존 `CalculateSleepDeficitUseCase`의 누적 부채 데이터 활용
- 회복 모델: 지수 감소 (하루 추가 수면 1시간 기준 부채 50% 회복)
- 예측 변수: 현재 부채량, 최근 7일 평균 수면 시간, 목표 수면 시간
- Sleep 대시보드에 "회복까지 약 N일" 인사이트 카드
- 부채 0인 경우 "수면 균형 달성" 표시

## 선행 조건

- 없음 (기존 수면 부채 시스템 활용)

## 참고

- 수면 부채 회복은 선형이 아닌 지수적 — 완전 회복보다 80% 회복이 실용적 목표
