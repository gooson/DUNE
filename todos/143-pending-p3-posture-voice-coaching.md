# Phase 4D: 음성 코칭

## 설명

실시간 자세 분석 결과를 음성으로 피드백하여 화면을 보지 않고도 교정 가능.

## 주요 작업

1. **음성 피드백 엔진**
   - AVSpeechSynthesizer 기반 (MVP)
   - 교정 메시지 큐 관리
   - 반복 억제: 같은 교정 최소 5초 간격
   - 우선순위: 위험(즉시) > 주의(큐잉) > 정보(생략 가능)

2. **교정 메시지 정의**
   - 운동별 교정 메시지 세트
   - 3개 언어 지원 (en/ko/ja)
   - 간결한 명령형: "무릎 밖으로", "허리 펴", "더 내려가"

3. **타이밍 최적화**
   - 문제 감지 → 음성 출력 지연 < 500ms
   - 운동 phase 고려 (하강 중에만 깊이 교정, 상승 중엔 무릎 교정)

4. **Future: 사전 녹음 음성**
   - TTS 대비 자연스러운 음성
   - 오디오 에셋 번들

## 착수 조건

- Phase 4B 완료 (폼 판정이 있어야 교정 메시지 생성 가능)
- Phase 4C는 독립 (병렬 착수 가능)

## 검증 기준

- [ ] 감지 → 음성 출력 지연 < 500ms
- [ ] 반복 억제 정상 동작 (5초 이내 동일 메시지 미발화)
- [ ] 운동 중 음성이 다른 오디오(음악)와 공존
- [ ] 3개 언어 교정 메시지 커버리지
