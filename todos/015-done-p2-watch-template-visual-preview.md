# Watch 템플릿 카드 시각적 프리뷰 강화

## 현재 상태

TemplateCardView가 장비 아이콘 strip을 보여주지만, 전체 레이아웃이 여전히 기본적.
브레인스톰에서 계획한 풍부한 프리뷰(세트/렙 요약, 예상 소요시간, 난이도)는 미구현.

## 목표

- 템플릿 카드에 총 세트 수, 예상 소요시간 표시
- 운동 순서 미리보기 (아이콘 + 이름 축약)
- 즐겨찾기/최근 사용 순서 정렬
- RoutineListView에서 카드 간 시각적 구분 강화
