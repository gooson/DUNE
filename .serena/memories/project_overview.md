# Health project overview
- Purpose: iOS Health & Fitness app focused on condition/readiness analysis using HRV/RHR and related HealthKit data.
- Stack: Swift 6, SwiftUI, HealthKit, SwiftData, CloudKit, XcodeGen.
- Targets: iOS app (`DUNE`), watchOS app (`DUNEWatch`), widgets, unit/UI test targets.
- Structure: `DUNE/` app source, `DUNEWatch/` watch source, `Shared/` shared assets/resources, `DUNETests/` and `DUNEWatchTests/` unit tests, `DUNEUITests/` and `DUNEWatchUITests/` UI tests, `docs/` plans/brainstorms/solutions/reviews, `scripts/` build and test helpers, `todos/` work items.
- Process: Compound Engineering workflow (`/brainstorm -> /plan -> /work -> /review -> /compound`).
