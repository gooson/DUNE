# Brainstorm: 습관 전체 리스트 뷰 (아카이브 포함)

## Problem Statement

현재 습관 시스템에서 `isArchived = true`로 아카이브된 습관은 `@Query` 필터에 의해 완전히 숨겨지며, 이를 다시 확인하거나 복원할 수 있는 UI가 없다. 사용자가 예전에 생성한 습관이 보이지 않아 데이터가 사라진 것으로 오해할 수 있고, 아카이브된 습관의 기록도 접근 불가.

### 현재 동작

- `LifeView`의 `@Query`가 `!$0.isArchived` 필터 적용
- 습관 컨텍스트 메뉴에 "아카이브" 액션은 있지만 "복원" 경로 없음
- 아카이브된 습관의 히스토리도 접근 불가

## Target Users

- 장기 사용자: 습관을 만들었다가 아카이브한 뒤 다시 찾고 싶은 사용자
- 데이터 확인: 과거 습관 기록을 회고하고 싶은 사용자

## Success Criteria

1. 활성 + 아카이브된 모든 습관을 한 화면에서 확인 가능
2. 아카이브 습관을 복원하거나 히스토리를 열람 가능
3. 각 습관의 통계 (총 기록 횟수, 최장 스트릭, 생성일) 확인 가능

## Proposed Approach

### 진입 경로

"내 습관" 섹션 하단에 아카이브된 습관 개수를 표시하는 링크 추가:

```
[내 습관 리스트]
  ...
  ──────────────
  📦 아카이브된 습관 3개 보기   →
```

탭하면 전체 습관 리스트 화면으로 NavigationLink push.

### 전체 습관 리스트 화면

**필터 전환**: 상단에 Picker 또는 segmented control

```
┌─────────────────────────────────────┐
│  습관 관리                           │
│  [활성 (5)] [아카이브 (3)]           │ ← Segmented filter
├─────────────────────────────────────┤
│  🏃 운동        12회 · 🔥7일 스트릭  │
│  📖 독서         8회 · 생성 2025-12  │
│  🧘 명상        24회 · 🔥14일        │
│  ...                                │
└─────────────────────────────────────┘
```

**활성 탭**: 현재 활성 습관 + 통계 요약
**아카이브 탭**: 아카이브된 습관 + 복원/히스토리 액션

### 각 습관 행 정보

| 항목 | 설명 |
|------|------|
| 아이콘 | 습관 아이콘 (기존 `icon` 필드) |
| 이름 | 습관 이름 |
| 총 기록 횟수 | `HabitLog` count |
| 최장 스트릭 | 연속 달성 최대 기록 |
| 생성일 | `createdAt` 표시 |
| 상태 배지 | 활성/아카이브 (아카이브 탭에서는 생략 가능) |

### 아카이브 습관 액션

| 액션 | 설명 |
|------|------|
| 복원 | `isArchived = false` 설정, 활성 리스트로 이동 |
| 히스토리 | 기존 `HabitHistorySheet` 재사용하여 과거 기록 열람 |

## Constraints

- **레이어 경계**: ViewModel은 SwiftData를 직접 import하지 않음. View에서 `@Query` + `modelContext` 처리
- **기존 UI 비침범**: "내 습관" 섹션의 기존 동작은 변경하지 않음
- **성능**: 아카이브 습관이 많아도 LazyVStack 등으로 처리

## Edge Cases

| 케이스 | 처리 |
|--------|------|
| 아카이브 습관 0개 | 하단 링크 자체를 숨김 |
| 활성 습관 0개 | 활성 탭에 기존 empty state 재사용 |
| 아카이브 습관 기록 0건 | "기록 없음" 표시 |
| 습관 복원 후 정렬 | sortOrder 마지막에 추가 |

## Scope

### MVP (Must-have)

- 섹션 하단 "아카이브된 습관 N개 보기" 링크
- 전체 습관 리스트 화면 (활성/아카이브 탭 전환)
- 각 습관 행: 아이콘, 이름, 총 기록 횟수, 최장 스트릭, 생성일
- 아카이브 습관 복원 액션
- 아카이브 습관 히스토리 열람
- 다국어 (en/ko/ja)

### Nice-to-have (Future)

- 활성 습관에서도 통계 상세 (주간 달성률 차트)
- 습관 영구 삭제 (데이터 완전 제거)
- 습관 정렬 변경 (이름순, 생성일순, 기록 횟수순)
- 활성 탭에서 드래그 재정렬
