# Brainstorm: 앱 아이콘 & 스플래시 디자인

## Problem Statement

DUNE 앱의 iOS/watchOS 아이콘과 런치 스크린이 비어 있음. 앱의 브랜드 아이덴티티를 시각적으로 확립하여 첫인상과 앱스토어 가시성을 확보해야 함.

## Target Users

- 건강/피트니스에 관심 있는 20-40대
- 데이터 기반 자기 관리를 선호하는 사용자
- Apple Watch + iPhone 생태계 사용자

## Brand Direction

### 핵심 키워드
- **DUNE** = 사막의 모래언덕 = 자연의 힘, 광활함, 리듬, 변화
- 건강 데이터의 파동(HRV, 심박수)과 모래 언덕의 곡선이 연결
- 사막의 생존력 = 신체의 회복력/컨디션

### 브랜드 느낌
| 축 | 방향 |
|---|---|
| Warm ← → Cool | Warm (사막의 따뜻함) |
| Organic ← → Geometric | Organic 쪽 (자연 곡선) |
| Playful ← → Serious | 중간~약간 Serious (데이터 신뢰감) |
| Minimal ← → Rich | Minimal (깔끔한 추상) |

## 앱 아이콘 컨셉 후보

### A. 모래 언덕 파형 (Dune Wave)
```
┌──────────────┐
│              │
│   ╱╲    ╱╲  │
│  ╱  ╲  ╱  ╲ │
│ ╱    ╲╱    ╲│
│╱            ╲│
│              │
└──────────────┘
```
- 모래 언덕의 곡선 = HRV/심박수 파형의 이중 의미
- 2-3개의 부드러운 웨이브 라인
- 다크 배경에 포인트 컬러 그라디언트 적용 가능
- **장점**: DUNE 이름과 직관적 연결, 건강 데이터 파형 이중 의미
- **단점**: 파형 아이콘이 오디오/음악 앱과 혼동 가능

### B. 추상 모래 입자 (Sand Particles)
```
┌──────────────┐
│     . ·      │
│   ·  ·  .    │
│  .  ●  ·  .  │
│   ·  .  ·    │
│     ·  .     │
└──────────────┘
```
- 중심에서 퍼져나가는 입자 패턴
- 데이터 포인트/산점도 느낌도 겸함
- **장점**: 독특하고 추상적
- **단점**: 작은 사이즈에서 디테일 손실

### C. 지형 등고선 (Topographic)
```
┌──────────────┐
│  ╭─────╮     │
│ ╭┤     ├╮    │
│╭┤│  ●  │├╮   │
│╰┤│     │├╯   │
│ ╰┤     ├╯    │
│  ╰─────╯     │
└──────────────┘
```
- 지형도/등고선 패턴으로 사막 언덕 표현
- 동심 곡선이 건강 데이터의 레이어/depth를 연상
- **장점**: 고급스럽고 독특함, 작은 사이즈에서도 인식 가능
- **단점**: 지도 앱으로 오인 가능

### D. 추상 태양 + 언덕 실루엣 (Desert Horizon)
```
┌──────────────┐
│      ○       │
│    ╱    ╲    │
│  ╱   ╱╲  ╲  │
│╱   ╱    ╲  ╲│
│  ╱        ╲  │
│╱            ╲│
└──────────────┘
```
- 지평선의 태양 + 모래 언덕 실루엣
- 일출 = 새로운 컨디션 시작의 의미
- **장점**: 직관적, 긍정적 이미지
- **단점**: 날씨/여행 앱과 혼동 가능

## 색상 전략

### 다크 배경 + 포인트 컬러

| 후보 | 배경 | 포인트 | 느낌 |
|------|------|--------|------|
| Desert Gold | #1A1A2E | #E8A838 → #D4763B | 사막 일몰, 따뜻함 |
| Sand Amber | #0D0D0D | #C9956B → #8B6914 | 모래색, 차분함 |
| Oasis Teal | #0F1923 | #2DD4BF → #14B8A6 | 오아시스, 생명력 |
| Pulse Orange | #111111 | #FF6B35 → #F72585 | 에너지, 심박 |

### 기존 앱 컬러와의 조화
- MetricHRV, MetricRHR 등 기존 metric 색상과 충돌하지 않는 포인트 컬러 선택
- AccentColor를 아이콘 포인트 컬러로 통일하면 브랜드 일관성 확보

## 스플래시 (런치 스크린)

### 방향: 심플 로고
- 다크 배경 (앱 아이콘 배경색과 동일)
- 중앙에 앱 아이콘 심볼만 배치
- 텍스트 없음 (Apple HIG 권장: 런치 스크린에 텍스트 지양)
- iOS 26 Liquid Glass 효과가 아이콘에 자동 적용되므로, 런치 스크린은 최대한 미니멀하게

### 기술 구현
- `Info.plist` 기반 LaunchScreen (UILaunchScreen key)
- `UIColorName`: 앱 배경색 지정
- `UIImageName`: 중앙 심볼 이미지 (optional)
- 또는 기존 LaunchScreen.storyboard 유지

## 플랫폼별 요구사항

| 플랫폼 | 사이즈 | 참고 |
|--------|--------|------|
| iOS | 1024x1024 (단일) | iOS가 모든 사이즈 자동 생성 |
| watchOS | 1024x1024 (단일) | 원형 마스크 적용됨 — 가장자리 여백 고려 |
| App Store | 1024x1024 | 투명 배경 금지, 모서리 라운딩 자동 적용 |

### watchOS 특수 고려
- 원형으로 잘리므로 모서리에 중요 요소 배치 금지
- 아이콘 중심부에 핵심 심볼 집중 배치
- 작은 사이즈(40x40pt)에서도 인식 가능해야 함

## Constraints

### 기술적 제약
- 이미지 에셋 제작은 외부 도구 필요 (Figma, Sketch, Illustrator 등)
- Claude Code에서 직접 픽셀 이미지 생성 불가
- SVG → PNG 변환 또는 직접 디자인 필요

### 접근 방법
1. **컨셉/방향 확정** (이 brainstorm)
2. **디자인 제작** — 방법 선택:
   - a) Figma/Sketch에서 직접 디자인
   - b) AI 이미지 생성 도구 활용 (DALL-E, Midjourney 등)
   - c) 전문 디자이너에게 의뢰
3. **에셋 통합** — xcassets에 이미지 배치

## Decisions

### 아이콘 컨셉: A. Dune Wave
- 모래 언덕의 부드러운 곡선 2-3개가 HRV/심박수 파형의 이중 의미를 전달
- 추상적이면서도 DUNE 이름과 직관적으로 연결
- 다크 배경 위에 Sand Amber 그라디언트 웨이브

### 포인트 컬러: Sand Amber
- 배경: `#0D0D0D` (순수 다크)
- 포인트: `#C9956B` → `#8B6914` (코퍼 → 앰버 그라디언트)
- 모래의 질감, 차분하고 자연스러운 느낌
- AccentColor로도 통일하여 앱 전체 브랜드 일관성 확보

### 스플래시: 심플 로고
- 배경: `#0D0D0D`
- 중앙: Dune Wave 심볼만 배치
- 텍스트 없음
