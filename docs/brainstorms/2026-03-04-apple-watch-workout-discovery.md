# Brainstorm: Apple Watch Workout Discovery

## Problem Statement
애플워치에서 운동 종류를 찾을 때, 운동명 대신 기구 이름만 알고 있는 상황에서 원하는 운동을 빠르게 찾기 어렵다. 현재는 운동명이 헷갈리면 탐색 정확도가 떨어지고 진입 시간이 길어진다.

## Target Users
- 주요 사용자: 본인(Apple Watch로 운동을 기록하는 개인 사용자)
- 핵심 니즈: 운동명 기억이 불완전해도 원하는 운동을 정확하게 찾고 바로 시작하기

## Success Criteria
- 사용자가 원하는 운동을 5초 이내에 찾는다.
- 운동명 혼동 상황에서도 잘못된 운동 선택 비율이 낮다.
- 검색/탐색 과정에서 재시도 횟수가 줄어든다.

## Proposed Approach
- 운동 리스트에 카테고리 필터를 추가해 탐색 범위를 빠르게 좁힌다.
- 음성 입력 기반 검색을 제공해 텍스트 입력 부담을 줄인다.
- 운동명/기구명 간 매핑(동의어, 별칭)으로 검색 정확도를 높인다.

## Constraints
- 범위는 "검색 사용성 개선"에 한정한다.
- 별도 기술적/일정 제약 없음

## Edge Cases
- 사용자가 운동의 정확한 공식 명칭을 모르는 경우
- 기구 이름과 운동 이름이 1:1로 대응하지 않는 경우
- 발음/언어 차이로 음성 인식 결과가 부정확한 경우
- 카테고리 선택 후에도 후보가 너무 많은 경우

## Scope
### MVP (Must-have)
- 카테고리 필터 기반 운동 탐색
- 음성 입력으로 운동 검색
- 운동명 혼동 완화를 위한 기본 동의어/별칭 매핑

### Nice-to-have (Future)
- 개인 사용 패턴 기반 추천 카테고리/운동 우선 노출
- 최근 선택 운동 및 즐겨찾기 자동 상단 고정
- 검색 실패 로그 기반 동의어 사전 자동 확장

## Open Questions
- 카테고리 체계는 어떤 기준(부위/기구/운동 유형)으로 설계할지
- 음성 입력은 watchOS 기본 Dictation만 사용할지, 후처리 룰을 둘지
- 동의어/별칭 데이터 소스는 정적 테이블로 시작할지, 사용자 데이터로 확장할지
- 정확도 측정 지표를 어떤 이벤트로 수집할지(첫 선택 성공률, 재검색률 등)
