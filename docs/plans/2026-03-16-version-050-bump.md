# Version 0.5.0 Bump + What's New Catalog Update

## Summary

버전 0.4.0 → 0.5.0으로 올리고 What's New 카탈로그에 0.5.0 릴리스 항목 7개를 추가합니다.

## Affected Files

| 파일 | 변경 내용 |
|------|----------|
| `DUNE/project.yml` | MARKETING_VERSION 0.4.0 → 0.5.0 (5 targets) |
| `DUNE/Data/Resources/whats-new.json` | 0.5.0 release 엔트리 추가 (7 features) |
| `Shared/Resources/Localizable.xcstrings` | 15개 새 문자열 en/ko/ja 번역 추가 |
| `DUNETests/WhatsNewManagerTests.swift` | 0.5.0 파싱/ID/L10N 테스트 3건 추가 |

## Implementation Steps

### Step 1: project.yml 버전 범프
- `MARKETING_VERSION: "0.4.0"` → `"0.5.0"` (5곳 replace_all)

### Step 2: whats-new.json 업데이트
- 0.5.0 release 객체를 배열 최상단에 추가
- 7개 feature items (승인된 내용)

### Step 3: Localizable.xcstrings 번역 추가
- Python JSON 조작으로 15개 키 추가 (intro 1 + title 7 + summary 7)
- en/ko/ja 3개 언어

### Step 4: 테스트 추가
- 0.5.0 파싱 테스트
- feature ID 매칭 테스트
- L10N 키 비어있지 않은지 테스트

### Step 5: xcodegen 재생성
- `scripts/build-ios.sh`로 프로젝트 재생성 + 빌드 검증

## Test Strategy

- `WhatsNewManagerTests`에 0.5.0 전용 테스트 3건 추가
- JSON 파싱, feature ID 세트, L10N 키 검증
- 빌드 통과 확인

## Risks / Edge Cases

- xcstrings 수동 편집 시 JSON 구조 깨질 위험 → Python 스크립트 사용
- introKey와 feature 목록 불일치 위험 → 승인된 내용 그대로 사용
- SF Symbol 이름이 없는 경우 → iOS 17+ 심볼 확인
